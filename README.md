## Getting Started

Clone the repository.

```sh
git clone git@github.com:ythecombinator/eventio.git
```

`cd` into the directory.

```sh
cd eventio
```

Install the project dependencies:

```sh
yarn

# or

npm install
```

Start the development server:

```sh
yarn dev

# or

npm run dev
```
